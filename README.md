# 多彩卡片项目

这是一个卡片管理应用，使用Next.js、Tailwind CSS和NextAuth实现。

## 开发环境设置

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```
